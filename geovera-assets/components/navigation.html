Design Features

  • Dark Header: WIRED-style with #0B0F19 background
  • Bold Border: 4px green accent (#16A34A) at bottom
  • Typography: Georgia serif for logo (editorial authority)
  • Navigation: Uppercase links with 0.05em letter-spacing
  • Sharp Corners: NO border-radius on CTA buttons
  • Mobile-First: Hamburger menu on mobile with slide-out panel
  • Sticky: Header stays at top when scrolling
  • Accessible: 44px touch targets, clear focus states

Usage Example

<!-- WIRED-Style Dark Header -->
<header class="header">
  <div class="header-container">
    <a href="/" class="header-logo">
      <div class="header-logo-icon">G</div>
      <span class="header-logo-text">GeoVera</span>
    </a>

    <nav class="nav-desktop">
      <a href="#" class="nav-link active">Home</a>
      <a href="#" class="nav-link">Features</a>
      <a href="#" class="nav-link">Pricing</a>
      <a href="#" class="header-cta">Get Started</a>
    </nav>

    <button class="mobile-menu-toggle">
      <svg>...</svg>
    </button>
  </div>
</header>

<!-- Mobile Navigation -->
<nav class="nav-mobile" id="mobileNav">
  <a href="#" class="nav-link">Home</a>

Variants

The navigation system includes two main variants:

  • Dark Header (Default): WIRED-style black header with white text - Use for homepage and main sections
  • Light Header: White background with black text - Use for content pages and articles
Both variants maintain sharp corners (border-radius: 0) and the same layout structure
