Usage Example

  Get Started
</button>

<!-- Secondary Outline Button -->

  Learn More
</button>

<!-- Button with Loading State -->

  Loading...
</button>

<!-- Button with Icon -->

  <svg>...</svg>
  Add Item
</button>

<!-- WIRED-Style Uppercase Button -->

<!-- Disabled Button -->

</button>

Design Notes

  • NO border-radius: All buttons have sharp corners (border-radius: 0)
  • Conservative colors: Primary green (#16A34A), black (#0B0F19), minimal palette
  • WIRED inspiration: Bold, editorial, newsletter-style design
  • Typography: Inter font, bold weight (700), clean and modern
  • Hover effects: Subtle transforms and shadows for depth
  • Accessibility: Minimum 44px touch targets, clear focus states
