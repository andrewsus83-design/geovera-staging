# GeoVera Icon Library

A custom SVG icon library designed specifically for GeoVera's sharp, editorial brand identity.

## Quick Start

### Using Sprite Sheet (Recommended)

```html
<svg class="icon" aria-hidden="true">
  <use href="/geovera-assets/icons/icons-sprite.svg#dashboard"></use>
</svg>
```

### Basic CSS

```css
.icon {
  width: 20px;
  height: 20px;
  stroke: currentColor;
  fill: none;
  stroke-width: 2;
}
```

## Design Specs

- **Grid:** 24x24px
- **Stroke:** 2px (square caps, miter joins)
- **Style:** Sharp corners, no rounded edges
- **Colors:** Green (#16A34A), Black (#0B0F19)
