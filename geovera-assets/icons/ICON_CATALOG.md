## Usage Guide

### HTML
```html
<img src="/geovera-assets/icons/dashboard.svg" alt="Dashboard" width="24" height="24">
```

### CSS (Change Color)
```css
.icon {
  filter: brightness(0) saturate(100%) invert(50%) sepia(98%) saturate(2476%) hue-rotate(86deg) brightness(96%) contrast(93%);
  /* Above converts #0B0F19 to #16A34A (GeoVera green) */
}

/* Or use mask for color control */
.icon-green {
  -webkit-mask: url('/geovera-assets/icons/dashboard.svg') no-repeat center;
  mask: url('/geovera-assets/icons/dashboard.svg') no-repeat center;
  background-color: #16A34A;
}
```

### React/Vue Component
```javascript
import DashboardIcon from '@/geovera-assets/icons/dashboard.svg';

<DashboardIcon className="w-6 h-6 text-green-600" />
```
