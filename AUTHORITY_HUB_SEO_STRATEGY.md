## ğŸ¯ KEYWORD INTEGRATION

### On-Page Keyword Placement

**Required:**
1. **Title tag** (H1): Include primary keyword in first 60 characters
2. **Meta description**: Include keyword naturally
3. **First paragraph**: Use keyword in first 100 words (citation block)
4. **H2 headers**: Include keyword or variation in 2-3 headers
5. **Alt text**: Use keyword in 1-2 image alt tags
6. **URL slug**: Include keyword (e.g., /hub/brand-intelligence-tools)

**Optional (Natural):**
7. Body text: 1-2% keyword density
8. FAQ questions: Include keyword variations
9. Image captions
10. Internal link anchor text

---
